# Funtico Frontend JS SDK Demo

## How to Run

### Option 1: Node.js http-server
```bash
npm install -g http-server
http-server -p 3000
```

### Option 2: Python
```bash
# Python 3
python -m http.server 3000

# Python 2
python -m SimpleHTTPServer 3000
```

### Option 3: npx serve
```bash
npx serve -p 3000
```

### Option 4: PHP
```bash
php -S localhost:3000
```

Open your browser to `http://localhost:3000`
